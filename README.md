# Student Management System (Menu-Driven)

This is a menu-driven student management system written in C. It allows you to manage student records including their ID, name, age, course, and grade.

## Features
- **Add Student**: Add new student records.
- **View All Students**: Display all stored student records.
- **Search Student**: Find a student by their ID.
- **Delete Student**: Remove a student record by their ID.
- **Data Persistence**: Records are saved in a binary file (`students.bin`).

## How to Run (C)
1. Compile the C program:
   ```bash
   gcc student_management.c -o student_management
   ```
2. Run the application:
   ```bash
   ./student_management
   ```

## Menu Options
1. Add Student
2. View All Students
3. Search Student by ID
4. Delete Student by ID
5. Exit
